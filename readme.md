# golang-aws-lambda-terraform

### This project is about:

- Terraform
- AWS
- Golang
- Lambda
- Api Gateway

## Run
- Change api_key, secret_key on terraform/main.tf. You can get it from [AWS Users](https://us-east-1.console.aws.amazon.com/iamv2/home?region=us-east-1#/users)

`make build`

`make init`

`make plan`

`make apply`

`make destroy`
